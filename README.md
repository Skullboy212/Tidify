# Tidify

Tidify is a project that integrates a backend built with FastAPI and an Android app built with Flutter, supplemented with configuration files for seamless development and deployment.

## Project Structure

- **backend/**: Contains the FastAPI backend.
- **android/**: Contains the Flutter Android app.
- **config/**: Configuration files for various environments.

### Hi-Fi API Integration

This project leverages Hi-Fi API to provide rich features. Make sure to check the [API documentation](https://hifi-api.com/docs) for details on endpoints and integration strategies.
